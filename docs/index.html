Navigation SDK

The Navigation SDK provides high-precision route guidance, map rendering, and real-time positioning optimized for mobile environments.

Integrate advanced routing algorithms with minimal configuration. The SDK supports turn-by-turn navigation, route recalculation, and context-aware updates for changing traffic conditions.

Designed for performance and reliability, the system maintains accurate location tracking even in challenging environments such as urban canyons or low-signal regions.

Use the Navigator component to initiate navigation sessions and receive continuous updates about route progress, estimated arrival time, and maneuver instructions.
